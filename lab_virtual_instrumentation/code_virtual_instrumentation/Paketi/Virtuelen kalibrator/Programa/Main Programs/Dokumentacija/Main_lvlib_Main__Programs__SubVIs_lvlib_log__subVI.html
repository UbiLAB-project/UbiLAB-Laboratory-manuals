Main.lvlib:Main_Programs_SubVIs.lvlib:log_subVI.vi

This subVI is used to write the the data to csv or txt file.

Path

file path is the path name of the file. If file path is empty (default) or is <Not A Path>, the VI displays a dialog box from which you can select a file. Error 43 occurs if you cancel the dialog box.

Surname

Enter your surname

Lab

Enter the name of your lab

Instrument

Enter the name of the calibrated instrument.

True Value

The true value of the unit

Incremented

Incremented value of the unit

Error

Enter the relative error string.

Error In

error in describes error conditions that occur before this node runs. This input provides standard error in functionality.

    		status

    status is TRUE (X) if an error occurred before this node ran or FALSE (checkmark) to indicate a warning or that no error occurred before this node ran. The default is FALSE.

    		code

    code is the error or warning code. The default is 0.

    		source

    source specifies the origin of the error or warning and is, in most cases, the name of the node that produced the error or warning. The default is an empty string.

Name

Enter your name.

Error Out

error in can accept error information wired from VIs previously called. Use this information to decide if any functionality should be bypassed in the event of errors from other VIs.

Right-click the error in control on the front panel and select Explain Error or Explain Warning from the shortcut menu for more information about the error.
