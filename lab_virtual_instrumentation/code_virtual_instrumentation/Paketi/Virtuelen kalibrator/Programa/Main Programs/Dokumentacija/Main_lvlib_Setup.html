Main.lvlib:Setup.vi

This VI is used to SetUP the signal properties.

1.Start SetUP to view the signal

2. Save Values to save the values that you pick to signal_values FGV

3. Stop SetUP to stop the SetUP

5. Exit SetUp

Start Setup

Click Start, to start the SetUp od the signal.

Save Values

Click Save, to save the changes to the signal value FGV.

Exit Setup

Click Exit, to stop and close the program.

Amplitude

You can change the value of the signal amplitude. The amplitude is in (V, A, Ohm, W). After you click Save the value will be transferred to the respective unit depending on the value.

Hz Value

You can change the value of the signal frequency. The value is in Hz. After you click Save the value will be transferred to the respective unit depending on the value.

Offset

Pick Offset of the signal.

Phase

Pick the Phase of the signal

Waveform

Pick the Type of the waveform.

Stop SetUP

Click to stop the SetUP

Signal

Here the signal you picked will be shown. If the signal is too dense, you can use the tools on the bottom left angle to zoom in or out the signal.
