Main.lvlib:Glavna_Programa.vi

Main VI program.

1. After starting the program, click PowerUP to turn on the calibrator.

2. The program in Standby can pick the signal values with the use of the keyboard or SetUP mode.

The SetUP VI can configure can change the signal type, offset and phase.

3. Click Operate to go to Operate mode. First 2 Warning message will be shown, then the program will start with Operate mode.

4. Click Standby to go to standby again, then the program will stop with the DaQ and Operating mode.

5. Click PowerUP to sotp the program
