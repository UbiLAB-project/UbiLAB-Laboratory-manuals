Main.lvlib:Save.vi

This VI, is used saving the information to a log file.

1. First window opens. If the user clicks cancel the default log file will be picked. If wrong file is picked, dialog window will be generated.

2. If there is missing information dialog window will be generated.

3. Click save to save to the log file.

4. Click Exit to exit the VI

 Name

Enter the name of the user.

Lab

Enter the laboratory.

Instrument

Enter the instrument that was calibrated.

Save Log

Click to save the data and close the program.

Surname

Enter the surname of the user

Exit Progam

Click to exit the program
